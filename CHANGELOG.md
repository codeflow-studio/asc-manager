# Changelog

All notable changes to ASC Manager will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2025-01-01

### Added

- Multi-account App Store Connect management
- App overview with filtering and grouping
- Version management with build selection and distribution info
- In-app purchase and subscription product management with localizations
- Xcode Cloud builds page with workflow, action, and log viewing
- Express API proxy with JWT authentication and response caching
- Terminal-style log viewer for Xcode Cloud build logs
